Project Title : Book Explorer

This program is that Users can add or find information about a book in book files. 
Also, users are able to check the book information in the book file at random as the number of books they want.

The date : June 07, 2020

The author : Sangmean Hong, Aiden(Gookjin) Kim

Users can run this program by running a file with a "jar" extension on the command prompt. 
Users should type "java -jar filename.jar" on the command prompt.
